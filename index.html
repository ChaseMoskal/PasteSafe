pastesafe v0.13.0-dev

instantly encrypt as you type

your web browser is performing the encryption itself

whatever you enter into pastesafe never leaves your computer — there's no serverside work here, it's just a flat HTML page hosted on github pages

built on the web cryptography api:

  • aes-256-gcm
  • 128 byte initialization vector
  • hexadecimal output
  • built for chrome

pre-release warning

pastesafe is a work-in-progress experimental application — you may use this software at your own risk
