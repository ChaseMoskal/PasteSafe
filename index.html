PasteSafe
Real time web cryptography

Tool for encrypting and decrypting text on the fly.

Open source (github project), MIT License.

AES-256 GCM cipher. Hexadecimal output. First 32 hex digits represent the randomly generated IV.

Work in progress! I'm calling it v0.0.0.
Works in Chrome and Firefox. Currently broken in Internet Explorer (sorry).
